# Client Status Web App (Demo)

A demo dashboard web app for client-facing project status updates, built on the same core tech stack as the MVP AI site:

- Next.js 16 (App Router)
- React 19 + TypeScript
- Tailwind CSS 4
- ESLint 9 + eslint-config-next

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Included Demo Dashboards

- Portfolio overview metrics
- Project health and progress cards
- Villagers assigned to each project
- Developer workstream status matrix
- Next-step action queue by priority
